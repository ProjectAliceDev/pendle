A desktop designed for AliceOS

The Bendy is not GNOME (BinG) Desktop shell provides a quick and easy way of launching AliceOS applets from within any AliceOS project. Its inspiration draws from the GNOME project.

Screenshots

Add BinG

BinG is currently under development, so some features may not work properly. To install BinG, extract the ZIP file and move the contents to Applets/BinG/.

Alternatively, you can clone the repository by running git clone git@github.com:ProjectAliceDev/bing.git 'BinG' directly into the Applets folder.
