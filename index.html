Meet your new Ren'Py platform

AliceOS is an independent, open-source project that brings a virtual operating system into Doki Doki Literature Club! as a mod and to Ren'Py projects as a framework. AliceOS is also the core operating system used in Doki Doki: The Angel Returns.

Built with Applets

Applets make up the app ecosystem for AliceOS. Applets are how third-party developers add additional functionality to AliceOS and interact with users and players. Want to add a new messaging Applet to AliceOS? or maybe even a game? Applets make it possible.

Rich notifications

AliceOS comes with its own notification framework that allows developers and users to stay informed. Applets and system services alike use notifications to express ideas, display information, or even make a request. Notifications are also user-controlled, giving the user ultimate control over which notifications can be seen.

Privacy matters

Privacy and security matter. That's why AliceOS comes with its own security policy, provided under the GOBFADU policy. Files have integrity protection and are not allowed to be tampered with. Furthermore, we enforce a policy to ensure that only you give the consent for an applet's control to things like notifications, file system access, and system setting changes.

Let's get geeky

AliceOS provides an experience similar to Linux, complete with a Terminal and a UNIX-like shell. From basic filesystem commands to package installation, AliceOS provides a unique experience to games. Applets can harness the power of the Terminal and offer additional commands and tools through its command-line interface, perfect for any character deletion (we're looking at you, Monika).
